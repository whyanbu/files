1.  in terminal:
	git init

2.  in terminal:
    git add .

3.  in terminal:
    git commit -m "Initial commit"

4.  in github web page:
	Go to github.com → Sign in.
		Click “+” → New repository.
		Fill in:	Repository name (e.g., CS601_Project)
					Description (optional)
					Public (or Private)
					Do NOT check “Add a README, .gitignore, or license”
		Click Create repository.

5.  in terminal:
	git remote add origin https://github.com/whyanbu/CS601_Project.git

6.  in terminal, Verify: 
	git remote -v

7.  in terminal:
	git branch -M main

8.  in terminal:
	git push -u origin main
	or 
	git push -u origin main --force (if error encountered)
